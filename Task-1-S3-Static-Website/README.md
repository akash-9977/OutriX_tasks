# Task 1: Deploy a Static Website on AWS S3

## Project Overview
This project demonstrates how to deploy a static website using Amazon S3.

## Tools Used
- AWS S3
- AWS Console
- HTML
- CSS
- JavaScript

## Steps Performed
1. Created a static website using HTML, CSS, and JavaScript.
2. Created an Amazon S3 bucket.
3. Enabled static website hosting.
4. Uploaded website files to the S3 bucket.
5. Configured public access using a bucket policy.
6. Tested the live website using the S3 website endpoint.

## Files Included
- index.html
- style.css
- script.js
- 404.html

## Learning
Through this task, I learned about S3 buckets, object storage, static website hosting, bucket policies, and public access configuration.
